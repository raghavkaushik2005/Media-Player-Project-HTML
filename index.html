Pure HTML Media Player

This media player allows users to play audio and video files using the built-in audio and video elements of HTML. The layout is styled with CSS, and a small amount of JavaScript is used to add interactivity. It provides a clean, beginner-friendly way to understand how multimedia can be embedded and controlled in a webpage.
